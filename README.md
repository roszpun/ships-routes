

## Project setup
```
git clone https://github.com/roszpun/ships-routes.git
```

```
cd ships-routes
```

```
yarn install
```

```
Remove .example from .env file. Provide google maps, api key
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
